## **📌 프로젝트 중점사항**

---
✔️ CI/CD를 적용하고 자동화된 빌드와 배포를 통해 개발의 생산성을 높히기 위해 노력했습니다.

✔️ 팀원과의 활발한 의사소통을 위해 노력 했습니다.

️️✔️ Gitflow 전략을 사용해 백엔드 역활 분활,협업을 하도록 노력 했습니다.


## **📌 주요 기능**

---
- 검색 기능
- 로그인
    - JWT
    - OAuth
        - 구글(학교 이메일 - @dsm.hs.kr)
- 비번 변경
- 이메일 인증
    - javamail
- 댓글, 대댓글 & JWT ➡ 세현
- 게시물 CUD ➡ 지민
- 회원가입, 로그인(Admin, User포함) ➡ 대현
